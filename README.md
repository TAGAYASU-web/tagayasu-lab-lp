Human Warmth Lab TAGAYASU の公式ランディングページです。

## 概要
- WordPress.com でホスト（カスタムHTML / blank テンプレート使用）
- ページID: 1489

## 主な機能
- Nukumol 心の温度診断（6段階シーズンカラー連動）
- ヒーロー温度カード（34.0°C → 36.03°C アニメ、シーズン名・グラデーションバー）
- スクロールダウン矢印（バウンスアニメ、ヒーロー底部中央）
- ページトップ戻るボタン（SVGシェブロン、スクロール400px超で表示）
- Scroll reveal アニメーション
